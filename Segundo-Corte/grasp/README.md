## Cómo compilar y ejecutar los programas

1. Abre una terminal y navega a esta carpeta (donde está el Makefile).

2. Para compilar todos los ejecutables:
	```sh
	make all
	```
	O puedes compilar uno específico, por ejemplo:
	```sh
	make grasp_pfsp3
	```

3. Para ejecutar el programa que desees, usa:
	```sh
	./grasp_pfsp    # Para el principal (main.cpp)
	./grasp_pfsp2   # Para main2.cpp
	./grasp_pfsp3   # Para main3.cpp
	./grasp_pfsp4   # Para main4.cpp
	./grasp_pfsp5   # Para main5.cpp
	```

4. Para limpiar los ejecutables generados:
	```sh
	make clean
	```
